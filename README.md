# NPC Shop Reroll with Influence

通常は影響力での再入荷に対応していない奴隷商人・調教師などのNPCに、影響力を消費してラインナップを即座に再入荷できるようにするElin MODです。

## 機能

- 奴隷商人・調教師などのNPCのウィンドウに「再入荷」ボタンが追加されます（ソートボタンの下）
- 影響力を 1 消費して、その場でラインナップを更新できます
- 影響力が不足している場合は再入荷できません
- 時間経過による自然補充の挙動は変わりません

## インストール

1. [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3726293987) からサブスクライブします
2. ゲームを起動するとMODが自動的にロードされます

## ビルド方法

```powershell
.\build.ps1
```
